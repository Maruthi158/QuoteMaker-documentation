About Quote Maker


This is a progressive web application. This application is to check their daily quotes which would be an inspiration to start their day. This application would also have flexibility where users would be able to check 10 quotes per day which would motivate them to complete their work on time, they also have the flexibility to modify the quote which displays on the screen and they can delete the quote if they do not like it.This app would be inspirational to complete things on time.
